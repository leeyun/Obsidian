# ADT
**Merge Sort(병합 정렬)** 이란 분할 정복(Devide and Conquer)과 재귀 알고리즘을 이용한 알고리즘이다.
```python
[6, 5, 3, 1, 8, 7, 2, 4]
```

```python
[6, 5, 3, 1] [8, 7, 2, 4]
```

```python
[6, 5] [3, 1] [8, 7] [2, 4]
```

```python
[6] [5] [3] [1] [8] [7] [2] [4]
```

이 상태에서 다시 두개 씩 합치면서 재배열 한다.
```python
[5, 6] [1, 3] [7, 8] [2, 4]
```

```python
[1, 3, 5, 6] [2, 4, 7, 8]
```

```python
[1, 2, 3, 4, 5, 6, 7, 8]
```

- 8-4-2-1 식으로 전반적인 반복의 수는 절반으로 줄어들기 때문에 ==O(logN)== 시간이 필요하며, 모든 값들을 비교해야 하므로 ==O(N)== 시간이 소요되므로 총 시간 복잡도는 ==O(NlogN)== 이다.
